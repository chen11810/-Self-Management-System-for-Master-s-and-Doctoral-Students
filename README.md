# 土壤生物膜与环境健康课题组 · 硕博研究生自我管理系统

## 📌 项目简介
本项目是为**华中农业大学资源与环境学院——土壤生物膜与环境健康课题组**定制开发的一款硕博研究生自我管理系统。系统集成了工作打卡、生活作息、科研进度、实验室耗材管理及团队数据实时共享等多种功能，旨在帮助研究生科学规划时间、提高科研效率，同时促进课题组内的团队建设与规范管理。

## 🚀 核心功能模块

### 1. ⏱️ 考勤与状态管理
- **每日心情记录**：快速选择今日心情，生成时间线。
- **打卡与计时**：提供“开始上班”、“休息一下”及“下班”功能，自动统计早、中、晚工作与休息时长。
- **定位与拍照打卡**：基于浏览器 Geolocation API 与摄像头，限定在“资环楼”周边 100m 范围内进行精确的定位与拍照打卡。

### 2. 📚 科研与任务管理
- **待办清单 (To-Do List)**：便捷记录并勾选待办任务。
- **文献与实验记录**：量化统计文献阅读篇数与实验次数。
- **专注模式**：帮助研究生在沉浸式科研时屏蔽干扰。
- **一键本周总结**：根据一周的工作时长、文献阅读数、完成任务数，一键生成本周的文本回顾。
- **实验工具箱**：内置实验数据速记与实验多路计时器。

### 3. ☕ 健康与生活追踪
- **摄入记录**：精准记录咖啡因摄入量（支持多品牌与品类）及每日饮水进度。
- **运动记录**：提供力量训练、球类、跑步、游泳等分类，并可计算热量消耗。
- **排出记录**：一键式入厕打卡，关注久坐研究生的身体健康。

### 4. 🧪 实验室综合协作
- **试剂耗材备忘录**：多人协作记录试剂存量状态（充足、偏少、需购）及存放位置。
- **卫生值班表**：可视化查看与安排实验室卫生值班工作。

### 5. 👥 课题组看板与互动
- **实时数据大屏**：结合 Firebase 实时数据库，展示全组今日出勤情况、历史总打卡及本月出勤排行榜。
- **课题组公告栏**：管理员可发布普通、重要及组会通知。
- **科研成就系统**：游戏化（Gamification）设计，解锁诸如“破土而出”、“月度达人”、“文献大师”等称号，获取成就点数。

## 🛠️ 技术栈
- **前端页面与样式**：原生 HTML5 + Tailwind CSS v3（CDN引入，配置化开发）
- **图标与数据可视化**：Font Awesome + Chart.js
- **后端与云服务**：Firebase (Authentication / Realtime Database) 提供轻量级的数据同步支持

## 💻 部署与运行方法

1. **环境准备**：
   本项目为纯前端驱动架构（搭配 Firebase BaaS）。无需配置复杂的本地 Node/Python 后端环境。
2. **启动应用**：
   将源码下载到本地后，使用现代浏览器（推荐 Google Chrome, Microsoft Edge）直接双击打开 `index.html` 即可运行。
3. **权限设置**：
   - 首次使用“定位打卡”需允许浏览器的**位置信息 (Geolocation)**权限。
   - 使用“拍照验证”需允许浏览器的**摄像头 (Camera)**权限。
4. **个性化配置**：
   通过页面顶部的“更换主题背景”与左侧边栏的“修改信息/上传头像”进行个人档案的配置。填写名字后即可与团队数据进行同步。

## 🔒 隐私与数据说明
所有健康记录（如饮水、入厕、心情等）仅保存于用户本地浏览器的 `localStorage` 中；公共互动数据（课题组看板、定位打卡简要记录、公告等）将同步至课题组的专属 Firebase 云端数据库，确保个人隐私与团队管理的完美平衡。
